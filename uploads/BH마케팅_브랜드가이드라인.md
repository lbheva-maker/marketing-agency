## 1. 브랜드 철학

### 미션
> **"작은 사업도 제대로 된 마케팅으로 성장할 수 있습니다."**

### 비전
> 대한민국 중소기업·소상공인의 가장 믿음직한 마케팅 파트너

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **신뢰** | 과장 없이 실제 성과로 말합니다 |
| **전문성** | 데이터와 경험을 기반으로 판단합니다 |
| **접근성** | 어렵지 않고, 비싸지 않게 제공합니다 |
| **파트너십** | 고객의 성장이 곧 우리의 성장입니다 |

## 3. 로고

### 3.1 기본형 (워드마크)

```
BH마케팅
```

- 로고는 **워드마크(텍스트형)** 으로 구성합니다
- 영문 `BH`는 굵은 세리프 또는 산세리프 계열로 강조
- 한글 `마케팅`은 본문 폰트 계열로 균형 있게 배치
- 전체적으로 **좌우 정렬, 수평 배치**를 기본으로 합니다

### 3.2 로고 여백 규정 (Clearzone)

로고 주변에는 반드시 **로고 높이의 50% 이상**의 여백을 확보합니다.

```
  [여백]
  BH마케팅
  [여백]
```

### 3.3 로고 사용 가능 형태

| 형태 | 배경 | 용도 |
|------|------|------|
| 기본형 (네이비) | 흰색·밝은 배경 | 명함, 제안서, 웹사이트 |
| 반전형 (흰색) | 네이비·어두운 배경 | 배너, 유니폼, 판촉물 |
| 흑백형 (블랙) | 흰색 배경 | 팩스, 흑백 인쇄물 |

### 3.4 로고 사용 금지 사항 (Don't)

- ❌ 로고 비율 임의 변경
- ❌ 로고에 그림자·외곽선·그라디언트 효과 추가
- ❌ 규정 외 색상으로 변경
- ❌ 배경과 구분되지 않는 색상 사용
- ❌ 로고 회전 또는 기울임 적용

## 5. 타이포그래피

### 5.1 한글 폰트

| 용도 | 폰트 | 굵기 |
|------|------|------|
| 제목 (H1~H2) | Noto Sans KR | Bold (700) |
| 소제목 (H3~H4) | Noto Sans KR | Medium (500) |
| 본문 | Noto Sans KR | Regular (400) |
| 강조 본문 | Noto Sans KR | SemiBold (600) |

> 대안 폰트: 나눔고딕, 프리텐다드(Pretendard)

### 5.2 영문 폰트

| 용도 | 폰트 | 굵기 |
|------|------|------|
| 로고·제목 | Montserrat | Bold (700) |
| 본문 | Noto Sans | Regular (400) |

### 5.3 폰트 크기 체계 (웹 기준)

| 단계 | 크기 | 용도 |
|------|------|------|
| Display | 48px | 메인 히어로 제목 |
| H1 | 36px | 페이지 대제목 |
| H2 | 28px | 섹션 제목 |
| H3 | 22px | 소제목 |
| Body L | 18px | 리드 문장 |
| Body | 16px | 본문 기본 |
| Caption | 13px | 주석·출처 |

### 5.4 타이포그래피 규칙

- 줄간격(Line Height): 본문 기준 **1.6~1.8배**
- 자간(Letter Spacing): 제목 `-0.02em`, 본문 `0`
- 단락 간격: 본문 폰트 크기의 **1.5배 이상**
- 한 줄 최대 글자 수: **한글 35자 이내** (가독성 기준)

## 6. 톤 & 보이스

### 6.1 브랜드 보이스 정의

BH마케팅은 **"마케팅을 잘 아는 동네 형/언니"** 같은 목소리를 냅니다.
어렵고 복잡한 마케팅 용어 대신, 사장님들이 바로 이해할 수 있는 말로 이야기합니다.

### 6.2 톤 특성

| 특성 | 설명 |
|------|------|
| **친근함** | 딱딱하지 않게, 부담 없이 다가갑니다 |
| **명확함** | 핵심만 짧고 쉽게 전달합니다 |
| **진정성** | 과장하지 않고 사실 그대로 이야기합니다 |
| **격려** | 고객의 도전과 성장을 응원합니다 |

### 6.3 언어 사용 원칙

**✅ 이렇게 쓰세요**
- 쉬운 한글 우선 사용
- 짧고 간결한 문장 (한 문장 = 하나의 메시지)
- 고객 중심 표현 ("사장님", "고객님")
- 구체적인 숫자와 사례 활용

**❌ 이렇게는 쓰지 마세요**
- 불필요한 영어 약어 남발 (CTR, ROAS, KPI 등을 설명 없이 사용)
- 과장된 표현 ("무조건", "100% 보장", "업계 최고")
- 고압적이거나 권위적인 문체
- 애매하고 추상적인 표현

### 6.4 상황별 톤 예시

#### 광고 카피

| 구분 | 나쁜 예 | 좋은 예 |
|------|---------|---------|
| 서비스 소개 | "최고의 퍼포먼스 마케팅 솔루션" | "광고비 아끼면서 매출 올리는 방법, 알려드릴게요" |
| 상담 유도 | "지금 즉시 문의하십시오" | "궁금한 거 뭐든 편하게 물어보세요 😊" |
| 성과 표현 | "압도적인 ROI 달성" | "한 달 만에 방문자가 2배 늘었어요" |

#### SNS 게시물

```
✅ 좋은 예
사장님, 인스타 광고 어렵게 느껴지시죠?
사실 처음엔 다들 그래요.
오늘은 딱 3가지만 알면 되는 인스타 광고 기초 알려드릴게요 👇

❌ 나쁜 예
Meta Ads Manager를 통한 효율적인 CPM 최적화 전략 및
타겟 세그먼트 설정 방법론에 대해 안내드립니다.
```

#### 이메일 / 제안서

- 인사말: "안녕하세요, BH마케팅입니다" (간단하고 친근하게)
- 문체: 격식체 유지하되 딱딱하지 않게
- 마무리: "궁금하신 점은 언제든지 연락 주세요"

## 7. 이미지 & 사진 스타일

### 7.1 사진 방향성

| 항목 | 기준 |
|------|------|
| 분위기 | 밝고 자연스러운 실제 현장 느낌 |
| 주인공 | 실제 소상공인·사업주·직원 |
| 구도 | 여백 충분히, 복잡하지 않게 |
| 색감 | 과도한 보정 없이 자연스러운 채도 |

### 7.2 사용 권장 이미지

- ✅ 실제 매장·사무실 환경 사진
- ✅ 사람 중심의 따뜻한 분위기
- ✅ 실제 데이터·성과 화면 캡처
- ✅ 밝고 깔끔한 배경의 제품 사진

### 7.3 사용 금지 이미지

- ❌ 과도하게 연출된 스톡 사진 (너무 완벽해 보이는 이미지)
- ❌ 어둡고 무거운 분위기의 사진
- ❌ 저해상도 이미지 (웹 최소 72dpi, 인쇄 최소 300dpi)
- ❌ 저작권 미확인 이미지

### 7.4 아이콘 & 일러스트

- 라인 계열 아이콘 우선 사용 (Filled 아이콘은 포인트에만)
- 컬러는 브랜드 컬러 시스템 내에서만 사용
- 한 디자인 내에서 아이콘 스타일 통일

## 8. 적용 사례 (Do / Don't)

### 명함

```
✅ DO
- 흰색 배경에 BH Navy 로고
- 이름·직책·연락처 명확하게 배치
- 여백 충분히 확보

❌ DON'T
- 배경에 복잡한 패턴 사용
- 여러 가지 컬러 혼용
- 작은 폰트로 정보 빽빽하게 채우기
```

### SNS 피드 (인스타그램)

```
✅ DO
- 그리드 일관성 유지 (3~4색 팔레트 고정)
- 텍스트는 이미지 내 최소화
- 브랜드 컬러 배경 + 흰색 텍스트 카드형

❌ DON'T
- 매번 다른 스타일의 디자인 사용
- 텍스트가 사진을 가리는 구성
- 압축된 저화질 이미지 업로드
```

### 제안서 / 보고서

```
✅ DO
- 표지에 BH Navy 배경 + 흰색 로고
- 데이터는 그래프·표로 시각화
- 페이지 번호·날짜 반드시 표기

❌ DON'T
- 폰트 3가지 이상 혼용
- 근거 없는 수치 사용
- 글자만 가득한 슬라이드
```
